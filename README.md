# SpaceX - The Unreal Reality

> Mixed Reality (Virtual Reality + Augmented Reality), Android, iOS

## About

It is Diploma Final Year Project on Mixed Reality (AR+VR).
A space simulation that depicts the vision of Elon Musk SpaceX mission with AI featured technology that uses AR and VR technology. 
It is developed in cross-platform environment i.e. on Android and iOS.

## Features

- Real world Simulation with immersive experience and Interactive Modelling with 3D objects.
- 360-degree video streaming experience with 360Â° Surround Sound.
- Coordination/combination of Augmented Reality and Virtual Reality.
- Realistic Virtual World with Unreal Experience.
- Creating Interactive Simulation by virtual controller or mobile as a controller.
- Gaze and Virtual Interactions.
- Speech Recognition.
- This provides Simulation-Based Learning, Gesture Recognition and Heads-up Display.


## Technology Used
- **Backend**
    - C#
    - AI Algorithms
    
- **Frontend**
    - App : Android & iOS
    
- **Development tools and SDKs**
    - Unity 3D
    - Sketchup Pro   
    - Mono Develop and Microsoft VS
    - Vuforia (AR plugin)
    - Google Cardboard (VR plugin)
    - Android and iOS Xcode SDK
    - IBM Watson
